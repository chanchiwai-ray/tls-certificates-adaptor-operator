# 001. Add dependency, create `constants.py`, rename charm class

## What

Add `charmlibs-interfaces-tls-certificates` to the `dependencies` list in `pyproject.toml` and regenerate `uv.lock`. Create `src/constants.py` with the project-wide string constants defined in the spec (Juju Secret label prefix, relation names, CSR hash algorithm). Rename the `Charm` class to `TLSCertificateAdaptorCharm` in `src/charm.py` and update the corresponding reference in `tests/unit/test_base.py`.

## Why

The dependency is required to use `TLSCertificatesRequiresV4` and `CertificateRequestAttributes` from charmlibs (spec: New-Interface Relation Data Format). Named constants prevent magic strings scattered throughout the codebase (general-coding-style instructions). The class rename aligns with Juju charm naming conventions.
